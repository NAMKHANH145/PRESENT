# 2nd Anniversary Love Journey

A beautiful interactive website celebrating 2 years of love and memories.

## Features

- Interactive login screen with custom authentication
- Loading screen with romantic messages
- "Yes or No" interaction game
- Timeline of special moments with photos
- Personalized letter section
- Special video surprise
- Background music and visual effects

## How to Run Locally

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git
```

2. Navigate to the project directory:
```bash
cd YOUR_REPOSITORY_NAME
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

## Build for Production

To build the project for production:
```bash
npm run build
```
